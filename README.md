# ChemStore

برنامج إدارة محل كيماوي وأسمدة ومبيدات.

## المميزات

- إدارة الأصناف
- إدارة العملاء
- المبيعات
- المشتريات
- الخزنة
- المصروفات
- التقارير
- النسخ الاحتياطي
- يعمل بدون إنترنت
- قاعدة بيانات SQLite

## التقنيات

- C#
- WPF (.NET 8)
- Entity Framework Core
- SQLite
